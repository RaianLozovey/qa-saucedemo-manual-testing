# 🧪 QA Portifolio - SauceDemo Manual Testing

## 📖 Sobre o projeto

Este repositório faz parte do meu portfólio de estudos em Quality Assurance (QA).

O objetivo deste projeto é aplicar na prática conceitos de testes manuais utilizando o sistema SauceDemo, simulando atividades realizadas por um Analista de QA.

Durante os testes foram criados casos de teste, executados testes funcionais e exploratórios, registradas evidências e documentados bugs encontrados.

---

## 🎯 Objetivos

- Planejar testes
- Criar casos de teste
- Executar testes funcionais
- Executar testes exploratórios
- Registrar bugs
- Documentar evidências

---

## 🔎 Escopo dos testes

- Login
- Produtos
- Carrinho
- Checkout

---

## 🛠 Técnicas utilizadas

- Testes Funcionais
- Testes Exploratórios
- Validação de Fluxos
- Registro de Bugs
- Documentação de Evidências
